# Hangman Game

## Description

This is a simple command-line Hangman Game developed in Python.

The program randomly selects a word from a predefined list, and the player must guess the word one letter at a time. The player has a limited number of incorrect guesses before the game ends.

## Features

* Random word selection
* Letter-by-letter guessing
* Input validation
* Repeated guess detection
* Win and lose conditions
* User-friendly console output

## Technologies Used

* Python
* Random Module

## Concepts Used

* Lists
* Loops
* Conditional Statements
* String Manipulation
* User Input/Output

## How to Run

1. Make sure Python is installed.
2. Save the file as `hangman.py`.
3. Open a terminal in the project folder.
4. Run:

```bash
python3 hangman.py
```
